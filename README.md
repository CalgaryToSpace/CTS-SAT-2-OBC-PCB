# CTS-SAT-2-OBC-PCB

This is the design repository for the CTS-SAT-2 onboard computer (OBC) PCB.

This OBC is host to an STM32 and several flash chips, and interfaces with all other electrical components on the 3U CubeSat.

## Repository Contents

* OBC PCB (KiCad; in progress) 
* STM32CubeIDE Pinout/IOC configuration (not yet done)
